## ✨ Overview

**Personal Portfolio Website** is a fully responsive web application that showcases the developer’s work, education, digital awareness, and writings. The website emphasizes interactive UI/UX elements like animated headings, gradient text, typing animations, dynamic sliders, and immersive moving backgrounds.

The contact section integrates **Google Forms backend** for seamless submission with regex validation. The project uses modular architecture with **low coupling and high cohesion**, and each section has fallback logic to maintain functionality if global files fail.

---

## 🚀 Features

* **Responsive Design:** Works on all devices using `clamp()` and media queries.
* **Independent Home Page:** Loads independently without global JS or CSS.
* **Typing Animation:** Showcases skills dynamically.
* **Gradient Text & Animated Headings:** Modern aesthetic appeal.
* **Home & Work Sections:**

  * Moving backgrounds for immersive experience.
  * Project cards with dynamic content.
  * Minimalist image slider with hover effects, transitions, and numbered navigation dots.
  * Optional **Download SRS** button for select projects.
* **About & Insights Section:**

  * Interactive image responding to cursor movement.
  * Education and work experience cards with links to **HackerRank**, **LinkedIn**, **LeetCode**.
  * Digital awareness summary with **Read More / Read Less** toggles.
  * Writings section featuring first novel with curly underline and gradient animations.
* **Contact Section:**

  * Dark mode styling for eye comfort.
  * Custom form integrated with **Google Forms backend**.
  * Regex validation for all inputs.
* **Global Assets:** Only `script-global.js` and `style-global.css` shared to reduce redundancy.
* **Fallbacks:** Independent modules include minimal backup code if global files fail.
