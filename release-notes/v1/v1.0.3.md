## ✨ Overview

This release integrates **SoundScript** — a fully client-side, modular web music player built with Vanilla JS — into the portfolio showcase.  
The update introduces **centralized documentation linking** using GitHub's raw file service, reducing local bloat and ensuring visitors always access the latest project specs.  
The portfolio remains **fast, responsive, and modular**, with new visual content and improved maintainability.
